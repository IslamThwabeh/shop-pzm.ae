Color System (CSS Variables)
Glass-Morphism (apply to navbar, cards, panels)

Typography & Spacing

Navbar
position: sticky; top: 0; z-index: 1000
Glass-morphism background (semi-transparent white + blur)
72px height desktop, 60px mobile
Nav links: gray-600, hover to dark with 2px green underline animation (width: 0 → 100%)
CTA button: green pill (border-radius: 50px, white text, var(--primary) bg)
IMPORTANT: backdrop-filter creates a CSS containing block. Any position: fixed children (like mobile menus) must use height: calc(100dvh - 60px) instead of bottom: 0, or be appended to <body> via JS.

Key Gotchas

backdrop-filter on navbar breaks position: fixed for children — plan for this in mobile menu
